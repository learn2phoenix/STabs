Privacy Policy for S-Tabs

Last Updated: November 14, 2024

Information Collection and Usage

S-Tabs collects and processes the following types of information to provide its core functionality:

1. Authentication Information

We collect and store:

  • Email addresses (@umd.edu domains only)
  • Encrypted password hashes
  • Authentication tokens

Purpose: To provide secure user authentication and maintain user sessions.

Retention: Authentication data is stored until account deletion.

2. Browser Data

We process:

  • Tab URLs and titles
  • Tab content for search indexing
  • Browser history (last 7 days, for search functionality)

Purpose: To enable tab search and management functionality.

Retention: Tab data is stored only during active sessions and is automatically cleared upon browser restart.

3. Device Information

We collect:

  • Device identifiers (for multi-device management)
  • Last access timestamps

Purpose: To enforce device limits and manage multi-device access.

Retention: Device information is retained until manual deregistration or account deletion.

Data Storage and Security

  • All data is stored locally on your device using Chrome's secure storage API
  • Authentication data is encrypted using industry-standard encryption
  • No data is shared with third parties
  • We do not use the data for any purpose other than providing the extension's core functionality

User Rights and Control

  • Users can delete their account and all associated data at any time
  • Users can deregister devices individually
  • Users can clear local search data through the extension settings

For questions about this privacy policy or data handling practices, please contact: learn2phoenix.umd@gmail.com
